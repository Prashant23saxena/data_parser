## Build Rules

- Preserve existing working Phase 1, Phase 2, and initial Phase 3 behavior.
- Use frozen contracts in `08-architecture/03-contracts/` as source of truth.
- Keep FastAPI routers thin; service behavior belongs in `backend/app/services/*`.
- Resolve workspace paths only through `backend-workspace-service`.
- Do not send table sample rows to LLMs by default.
- Pipeline service orchestrates; execution and Odoo sync stay delegated.
- Update tests with every behavioral change.
