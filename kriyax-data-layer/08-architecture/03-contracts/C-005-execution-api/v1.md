# Contract C-005: Execution API

**Version:** v1
**Status:** FROZEN
**Last updated:** 2026-05-03
**Type:** http-api
**Producer:** backend-execution-service via backend-api-routers
**Consumers:** frontend-code-workspace, frontend-pipelines, frontend-agent-panel

## Purpose

Save, list, open, and run local Python/pandas scripts with captured output and derived table registration.

## Shape

### POST `/api/execution/run`

Request:
```json
{"scriptName":"clean_orders.py","code":"df = load_table('raw_files.orders')\nprint(df.head())"}
```

Response 200:
```json
{"status":"success","scriptName":"clean_orders.py","scriptPath":"/workspace/scripts/clean_orders.py","stdout":"...","stderr":"","returnCode":0,"savedTables":["curated.clean_orders"]}
```

### GET `/api/execution/scripts`

Response 200:
```json
{"scripts":[{"name":"clean_orders.py","path":"/workspace/scripts/clean_orders.py","size":120,"updatedAt":"2026-05-03T00:00:00+00:00"}]}
```

### GET `/api/execution/scripts/{scriptName}`

Response 200:
```json
{"name":"clean_orders.py","path":"/workspace/scripts/clean_orders.py","size":120,"updatedAt":"2026-05-03T00:00:00+00:00","code":"print('hello')"}
```

Errors:
- 400 `validation_failed`
- 404 `script_not_found`
- 500 `execution_failed`

## Examples

### Happy Path

Run a script that calls `save_table(df, "clean_orders")`; response includes `savedTables: ["curated.clean_orders"]`.

### Error

Missing table returns `status: "error"` with traceback in `stderr` and available table hint.

### Edge Case

Script names without `.py` are normalized to `.py`.

## Mock Specification

If `code` contains `raise` or `missing_table`, return `status: "error"` and non-empty stderr. Otherwise return success and echo script name. Script list mock returns two scripts.
