# Contract C-001: Shared Types

**Version:** v1
**Status:** FROZEN
**Last updated:** 2026-05-03
**Type:** shared-type
**Producer:** shared-contract-types
**Consumers:** all frontend and backend modules

## Purpose

Define canonical JSON shapes used across frontend TypeScript helpers, FastAPI request/response models, mock fixtures, and tests.

## Versioning Policy

Breaking field removals, type changes, enum changes, or renamed fields require a new major contract version. Additive optional fields may stay in v1.

## Shape

```ts
type ISODateTime = string;
type AppType = "text" | "integer" | "decimal" | "date" | "datetime" | "boolean";
type RunStatus = "running" | "success" | "error";

interface ErrorResponse {
  error_code: string;
  message: string;
  fields?: Record<string, string>;
}

interface CatalogColumn {
  name: string;
  sourceName: string;
  type: AppType | string;
  nullable?: boolean | "unknown";
  description?: string | null;
}

interface CatalogTable {
  qualifiedName: string;
  schema: string;
  tableName: string;
  rowCount: number;
  columnCount: number;
  createdAt: ISODateTime;
  source: {
    kind: "file" | "odoo" | "python" | "external" | string;
    fileName?: string;
    filePath?: string;
    connectionName?: string;
    model?: string;
    scriptName?: string;
    scriptPath?: string;
  };
  columns: CatalogColumn[];
  warnings?: string[];
}

interface PipelineRun {
  id: string;
  pipelineId: string;
  pipelineName: string;
  scriptId: string;
  trigger?: "manual" | "schedule";
  status: RunStatus;
  startedAt: ISODateTime;
  endedAt?: ISODateTime | null;
  durationMs?: number | null;
  stdout: string;
  stderr: string;
  returnCode: number;
  savedTables: string[];
}
```

PII/secrets:
- `apiKey` is secret and must never be returned unmasked.
- DataTable row values may contain client PII; do not include sample rows in agent prompts unless explicitly approved by C-008.

## Examples

### Happy Path

`CatalogTable.source.kind = "file"` with `columns` populated from an imported CSV.

### Error

```json
{"error_code":"validation_failed","message":"Column name is duplicated.","fields":{"columns.1.targetName":"duplicate"}}
```

### Edge Case

Unknown database type maps to `CatalogColumn.type = "text"` or an implementation-specific string; consumers must display it without crashing.

## Mock Specification

Mock shared types by using the example objects in the downstream contract files. Unknown optional fields should be ignored by consumers.
