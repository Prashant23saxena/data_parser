# Contract C-003: Catalog API

**Version:** v1
**Status:** FROZEN
**Last updated:** 2026-05-03
**Type:** http-api
**Producer:** backend-catalog-service via backend-api-routers
**Consumers:** frontend-catalog, frontend-code-workspace, frontend-agent-panel

## Purpose

Expose table registry, column metadata, search, and row preview for humans, code workspace context, and agent schema context.

## Shape

### GET `/api/catalog/tables`

Query params:
- `q?: string`
- `source?: string`

Response 200:
```json
{"tables":[{"qualifiedName":"raw_files.orders","schema":"raw_files","tableName":"orders","rowCount":2,"columnCount":3,"createdAt":"2026-05-03T00:00:00+00:00","source":{"kind":"file","fileName":"orders.csv"},"columns":[{"name":"customer","sourceName":"Customer","type":"text"}]}]}
```

### GET `/api/catalog/tables/{schema}/{tableName}/preview`

Query params:
- `limit?: number` default 20, max 100

Response 200:
```json
{"schema":"raw_files","tableName":"orders","columns":["customer","amount"],"rows":[{"customer":"Acme","amount":100.5}]}
```

### GET `/api/catalog/schema-context` target

Response 200:
```json
{"tables":[{"qualifiedName":"raw_files.orders","columns":[{"name":"customer","type":"text"}]}],"sampleRowsIncluded":false}
```

Errors:
- 404 `table_not_found`
- 500 `catalog_unavailable`

## Examples

### Happy Path

Code Workspace loads `/api/catalog/tables` and lists `raw_files.orders` as available.

### Error

Previewing a deleted table returns:
```json
{"error_code":"table_not_found","message":"Table raw_files.orders was not found."}
```

### Edge Case

Wide table previews return all column names but only the first `limit` rows.

## Mock Specification

Return two tables: one file source and one python source. Preview returns two rows. If tableName starts with `missing`, return 404.
