## What this WP delivers

Odoo imports can run incrementally using a persisted cursor and upsert into an existing DuckDB table.

## Inputs

- Data model entity OdooSyncCursor

## Hard dependencies

## Soft dependencies

## Build steps

1. Add sync cursor metadata storage.
2. Validate cursor field exists on selected model.
3. Fetch records where cursor field is greater than last cursor.
4. Upsert into target table using Odoo `id`.
5. Update cursor only after successful write.
6. Add tests for insert, update, no changes, missing cursor field.
