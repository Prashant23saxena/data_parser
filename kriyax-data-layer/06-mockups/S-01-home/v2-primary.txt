Primary click paths:
- Start file import -> S-03 File Import Wizard
- Open Odoo setup -> S-04 Odoo Import Workspace
- Write Python -> S-09 Code Workspace
- Open catalog / table rows -> S-05 Catalog or S-06 Table Detail
- Review run / failed badge -> S-13 Pipeline Run Detail
- Open saved scripts -> S-11 Saved Scripts
- Open storage settings -> S-14 Storage Settings

Important alternate states:
- Empty workspace: quick actions remain, health cards show zeroes, recent tables/scripts/runs panels show first-use empty messages.
- Healthy: active failure alert collapses into "No active failures"; failed count is 0.
- Loading: skeleton cards for health and recent work; quick actions still visible but history tables disabled.
- History unavailable: pipeline health shows a recoverable inline error with [Retry] and link to Pipelines.
