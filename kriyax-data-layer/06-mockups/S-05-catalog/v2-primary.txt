Important alternate states:
- Empty catalog: import/connect/code quick actions, no table registry rows.
- Loading: skeleton table rows and disabled preview drawer.
- Error: inline "Could not load catalog" with retry and storage settings link.

Action meanings:
- Details: opens the Table Detail screen with metadata, columns, preview, export, and manage-table actions.
- Preview rows: opens the inline preview drawer only; it does not mean full data browsing.
- Use in Python: opens Code Workspace and inserts/loads the table through `load_table("table_name")`.
- Export CSV: opens the export modal.
