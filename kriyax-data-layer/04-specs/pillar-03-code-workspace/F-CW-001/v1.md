# Spec — F-CW-001: Python code editor

**Version:** v1
**Status:** FROZEN
**Last updated:** 2026-05-02
**Pillar:** pillar-03-code-workspace
**Feature ref:** F-CW-001
**Dependencies:** none

## Summary
In-browser Python code editor with syntax highlighting, line numbers, auto-indent, and a Run button. Executes Python scripts server-side.

## States

### empty
Empty editor with placeholder: "# Write your Python/pandas code here." Run button visible but disabled (no code).

### editing
User typing code. Run button enabled. Unsaved indicator shown if content modified.

### running
User clicked Run. Code sent to backend for execution. Spinner: "Running script…" Editor read-only during execution.

### completed
Execution finished. Output visible in panel below (F-CW-005). Editor writable again.

### error:runtime
Script threw a Python exception. Error traceback shown in output panel. Editor writable.

### error:timeout
Script ran longer than timeout (default: 60 seconds). Killed.
Message: "Script timed out after 60 seconds. Optimize your code or increase the limit."

## Inputs
| Field | Type | Required | Default | Validation |
|---|---|---|---|---|
| code | string | yes | empty | valid Python syntax (validated on run, not on type) |
| timeout | int | no | 60 | 5-300 seconds |

## Transitions
| From | To | Trigger | Side effects |
|---|---|---|---|
| empty | editing | user types code | none |
| editing | running | user clicks Run | code sent to backend |
| running | completed | execution returns successfully | stdout/output displayed |
| running | error:runtime | Python exception | traceback displayed |
| running | error:timeout | timeout exceeded | process killed |
| error:* | editing | user edits code | output preserved |

## Edge cases
- Infinite loop → timeout kills it
- Script imports not-installed package → error: "ModuleNotFoundError: No module named 'xxx'"
- Very large output (10k+ lines) → truncate at 5000 lines with "Output truncated"
- Multiple Run clicks in quick succession → debounce, only one execution at a time
- Code with syntax error → Python traceback on first line, shown clearly

## Acceptance checklist
- [ ] Type Python code → syntax highlighted correctly
- [ ] Click Run → output shown below
- [ ] Script with error → traceback displayed
- [ ] Infinite loop → timeout message after 60s
- [ ] Empty editor → Run button disabled
